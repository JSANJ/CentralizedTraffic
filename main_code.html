Import Dependencies¶

Dependencies:
numpy - for handling data types (mostly handled as numpy arrays)
matplotlib.pyplot - for plotting images
matplotlib.animation - for animating (creating the GIF)
CarClass - for data stored in each car and possible actions for cars
MapClass - for plotting data
time - for animation

constants:
area_definition - to be removed
constant_speed - normal speed in intersection
length of runway - road length before reaching intersection (for adjusting speed with constant acceleration)
intersection_x, intersection_y, intersection_w, intersection_h (x,y,w,h) coordinates of intersection on absolute map
car_width, car_length, assumed dimensions of cars
playback_speed - adjusted to speed up or slow down GIF playback

The directory is also defined in the PATH variable.
